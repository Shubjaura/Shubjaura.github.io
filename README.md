# Shubjaura.github.io

Personal portfolio website for Shubhkarman Jaura showcasing experience, projects, achievements, and artwork.

## Features

- Responsive single-page design
- Interactive project cards with flip animations
- Apple-style spotlight carousel for achievements
- Lightbox gallery for paintings
- Contact form with EmailJS integration

## Tech Stack

- HTML5, CSS3, JavaScript
- jQuery for DOM manipulation
- Typed.js for typing animations
- Owl Carousel for carousels
- EmailJS for contact form functionality

## Deployment

Simply push to GitHub and enable GitHub Pages in repository settings.
